# CAC
Config As Code, to build my config anywhere in a click

### Soft to install 
* Chocolatey
* Chrome
* Git
* Git Bash
* Launchy
* Putty
* Visual Studio

## Chocolatey
```
choco install launchy
choco install git
choco install vscode
choco install putty
choco install googlechrome
```
